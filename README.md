## Run Locally

1. Clone the project

```bash
  # using https 
  git clone https://github.com/pablocybernetic/survey.git

  # using ssh
  git clone git@github.com:pablocybernetic/survey.git

```

2. Go to the project directory

```bash
  cd survey
```

3. Install dependencies

```bash
  # using npm
  npm install
  
  # using yarn
  yarn
```
